# 🎯 Interactive Number Guessing Game

A simple and interactive number guessing game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Random number generation (1–20)
- Maximum 5 attempts
- Real-time feedback (Too high / Too low)
- Attempt tracking system
- Restart game functionality

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 How to Use
1. Enter a number between 1 and 20
2. Click "Guess"
3. Follow the hints to find the correct number
